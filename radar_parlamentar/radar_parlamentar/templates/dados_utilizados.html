    Extrações da base de dados do Radar

    Aqui você pode acessar a última extração realizada com todos os dados da base do Radar. Esses dados seguem o modelo de dados do Radar e estão disponibilizados no formato SQL, sendo extraídos periodicamente de nossa base de dados em PostgreSQL. Para utilizar esses dados, é preciso importá-los em alguma base de dados com suporte a SQL.

    Download da base completa do Radar aqui (atualizada em {{dumpdate}}).

    Ao disponibilizar aqui dados brutos oriundos de diferentes casas legislativas sob um formato unificado, esperamos que outras pessoas possam mais facilmente aplicar análises automatizadas sobre os dados das diferentes casas legislativas.
